# Frontend Tasks Collection

This repository contains small CSS and Bootstrap projects for practice and learning.

- Responsive Navbar
- Carousel/Slider
- Modal/Popup
- Cards Layout (with images, titles, and buttons)
- Accordion
- Alert/Notification Boxes (success, warning, error)
- Pricing Table (3-column responsive table)
- Grid Layout Examples (multi-column responsive layouts)
- Responsive Footer (links, social icons, contact info)
- Offcanvas Sidebar/Menu
- Tabs & Pills Navigation
- Tooltips & Popovers
